# Capturas de pantalla - Error alias no reconocido

## Ubicación de capturas

Si tienes capturas de pantalla del error, guárdalas en esta carpeta con nombres descriptivos:

- `error-terminal.png` - Captura del error en PowerShell
- `get-module-output.png` - Salida de `Get-Module -ListAvailable`
- `solucion-aplicada.png` - Captura después de aplicar la solución
- `verificacion-alias.png` - Salida de `Get-Alias wtranslator`

## Formato recomendado

- Formato: PNG o JPG
- Resolución: Legible (mínimo 1280px de ancho)
- Incluir fecha/hora si es posible
- Resaltar el mensaje de error relevante

## Referencia en documentación

Para incluir las imágenes en el documento principal, usa:

```markdown
![Descripción](./screenshots/nombre-archivo.png)
```

## Nota

La captura original del usuario mostraba el error en PowerShell con el módulo listado pero el alias no reconocido, lo cual confirmó el diagnóstico del problema.
