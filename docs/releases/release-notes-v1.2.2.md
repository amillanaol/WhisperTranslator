# WhisperTranslator v1.2.2

Esta versión es una actualización de mantenimiento que mejora la documentación de instalación, agregra información de citación y actualiza los metadatos del módulo.

## Novedades y Mejoras

### Documentación
- **Instrucciones de Instalación Mejoradas:** Se corrigieron las instrucciones de instalación en el README para apuntar a la ruta correcta del instalador (`.\module\Install-WhisperTranslator.ps1`).
- **Citación de Proyecto:** Agregado archivo `CITATION.cff` para proporcionar información estándar de citación del proyecto.

### Mantenimiento
- **Actualización de Metadatos del Módulo:** Se actualizaron los metadatos internos del módulo PowerShell en `WhisperTranslator.psd1`.
- **Sincronización de Versiones:** Versión del módulo sincronizada en todos los archivos de configuración.

## Instalación

Consulta la [Guía de Instalación](https://github.com/amillanaol/WhisperTranslator/tree/main/docs/instalacion).

## Uso Básico

Continúa usando el alias `wtranslator`:

```powershell
# Usar alias corto
wtranslator -m small -e mkv -d .

# Ver ayuda completa
wtranslator -h
```

Sin cambios.
