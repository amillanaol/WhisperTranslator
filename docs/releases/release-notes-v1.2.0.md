# WhisperTranslator v1.2.0

Esta versión introduce mejoras en la usabilidad y la documentación, destacando la adición de un alias para el comando principal y una estandarización de rutas de usuario en la documentación.

## Novedades y Mejoras

### Funcionalidades
- **Alias `wtranslator`:** Añadido alias `wtranslator` para `Invoke-WhisperTranslator`.

### Documentación
- **Notas de Release v1.1.5:** Añadidas notas de release para la versión v1.1.5.
- **Estandarización de Rutas:** Rutas de usuario reemplazadas con `$env:USERPROFILE` para portabilidad.
- **Actualización de Guías:** Guía de inicio rápido y ejemplos de uso en `README.md` actualizados.
- **Documentación de Versiones:** Añadida documentación sobre etiquetado de versiones y script de automatización.
- **Documentación de Releases:** Incorporada documentación y plantillas para gestión de releases.
- **Badge de Release:** Añadido badge de release al `README.md`.
- **Eliminación de Sección Obsoleta:** Retirada sección de estado obsoleta del `README.md`.
- **Actualización de URL del Badge:** Actualizada URL del badge de release de GitHub.

### Mantenimiento
- **Configuración de Markdownlint:** Actualizado `.markdownlint.json` para deshabilitar MD051.

## Instalación

Consulta la [Guía de Instalación](https://github.com/amillanaol/WhisperTranslator/tree/main/docs/instalacion).

## Uso Básico

Ahora puedes usar el alias `wtranslator`:

```powershell
# Usar alias corto
wtranslator -d ".\videos" -m small -e mkv

# Ver ayuda completa
wtranslator -Help
```
